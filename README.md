
# 📰 Cross News — מעקב שקרים ופייק ניוז בחדשות ישראל

אפליקציית אנדרואיד לניטור, ניתוח והשוואת כתבות חדשותיות ממקורות ישראליים מרכזיים. האפליקציה מזהה דיווחים דומים בין מקורות שונים, מנתחת הטיות אפשריות, ומאפשרת למשתמש להשוות בין כתבות באופן חזותי ובאמצעות AI.

---

## 🎯 מטרת האפליקציה

בעידן של ריבוי מקורות חדשות וזוויות סיקור שונות, קשה לצרכן הממוצע לזהות:

- **האם הידיעה מאומתת?** האם היא מופיעה גם במקורות אחרים, או שמא זהו דיווח בודד?
- **כיצד מקורות שונים מסקרים את אותו אירוע?** האם יש הבדלים משמעותיים בניסוח, בדגשים או בעובדות?
- **האם קיימת הטיה?** בזיהוי דפועים חוזרים של ניסוח רגשני, כותרות סנסציוניות וכו'.

האפליקציה נבנתה כדי לתת כלי שקיפות חדשותית בידי הציבור.

## 🔍 תכונות עיקריות

### 📡 איסוף חדשות ממקורות מרובים
האפליקציה משיגה כתבות בזמן אמת מהמקורות הבאים:

- **Ynet** — ynet.co.il
- **N12** — n12.co.il
- **כאן 11** — kan.org.il
- **ערוץ 14** — c14.co.il
- **ערוץ 13** — 13tv.co.il

### 🔗 זיהוי דיווחים דומים במקורות שונים
האפליקציה משתמשת ב-AI כדי לזהות האם אותו אירוע מדווח במקורות שונים, ומציגה תג אימות:

- ✅ **מאומת במספר מקורות** — הידיעה נמצאה ב-2+ מקורות
- ⚠️ **נמצא במקור נוסף** — הידיעה מופיעה גם במקור אחד נוסף
- ❌ **לא אומת במקורות אחרים** — דיווח בודד, כדאי לבדוק שוב

### 🤖 ניתוח והשוואה באמצעות AI
- **סיכום כתבה** — סיכום קצר באמצעות מודל שפה
- **השוואה בין כתבות** — ניתוח הבדלי ניסוח, דגשים וזוויות סיקור בין שני מקורות
- הניתוח מתבצע בעברית תקנית ובסגנון עיתונאי

### 📊 מסך השוואה ידנית
- בחירה ידנית של שתי כתבות להשוואה צד-אל-צד
- שלושה מצבי תצוגה: אנכי, אופקי, מסך מלא
- שמירת מצב גלישה ב-WebView גם במעבר בין פריסות

### 📋 לוג ניתוח
- מעקב אחר הסיבות שבגינן כתבה קיבלה דירוג מסוים
- חיפוש בתוך הלוג לפי כותרת, מקור או סיבה

### 🔖 מועדפים (סימנייה)
- שמירת כתבות למועדפים בלחיצת כפתור
- מסך מועדפים ייעודי לגישה מהירה לכתבות שמורות
- הכתבות נשמרות מקומית וזמינות גם ללא חיבור לאינטרנט

### 🎨 סגנון עיתונאי
- עיצוב בהשראת אפליקציות חדשות מקצועיות
- צבע אדום "breaking news" כצבע המותג
- טיפוגרפיה עם פונט סריפי לכותרות
- תמיכה מלאה ב-RTL (עברית)
- מצב כהה/בהיר אוטומטי לפי מערכת ההפעלה

## 📖 שימוש באפליקציה

### מסך ראשי — חדשות
- גלילה בפיד המאוחד מכל המקורות
- סינון לפי סטטוס אימות (הכל / נמצא / לא נמצא)
- סינון לפי מקורות ספציפיים
- לחיצה על כתבה פותחת את כרטיס הפרטים שלה

### כרטיס כתבה
- תצוגת פרטי הכתבה והמקור
- **הוספה למועדפים** — לחיצה על אייקון הכוכב מוסיפה את הכתבה למועדפים
- **פתיחה בדפדפן / WebView** — קריאת הכתבה המלאה
- **סיכום עם AI** — לחיצה על הכפתור המתאים
- **דיווחים דומים ממקורות אחרים** — מציג ידיעות דומות במקורות אחרים, עם אחוז דמיון

### מועדפים
- גישה דרך הכרטיסייה התחתונה באפליקציה
- כל הכתבות שנשמרו מוצגות ברשימה מסודרת
- ניתן להסיר כתבות מהמועדפים בכל עת
- הכתבות נשמרות מקומית וזמינות גם ללא חיבור לאינטרנט

### השוואה ידנית
1. בחלק התחתון, עבור למסך **"השוואה"**
2. לחץ על אייקון ההשוואה ובחר 2 כתבות
3. לחץ על כפתור "השווה"
4. במסך ההשוואה:
   - שנה פריסה (אנכי / אופקי / מסך מלא) בלחיצה על אייקון הפריסה
   - לחץ על ✨ לניתוח AI המשווה בין הכתבות

### השוואה אוטומטית
- מסך **"ניתוח השוואה"** מציג קבוצות של ידיעות דומות שנמצאו אוטומטית
- בחירה בקבוצה מאפשרת השוואה מהירה בין המקורות

### לוג ניתוח
- מציג עבור כל כתבה את הסיבות שבגינן קיבלה דירוג
- שימושי לניפוי באגים ולהבנת מנוע הניתוח

## 🏗️ ארכיטקטורה

האפליקציה בנויה בארכיטקטורה מודרנית של אנדרואיד:

- **Jetpack Compose** — UI הצהרתי
- **MVVM** — הפרדת לוגיקה ותצוגה
- **Hilt** — הזרקת תלויות
- **Room** — בסיס נתונים מקומי
- **Coroutines + Flow** — עבודה אסינכרונית
- **Navigation Compose** — ניווט בין מסכים


### שירותים חיצוניים

- **Jina Reader** (`r.jina.ai`) — חילוץ טקסט נקי מעמודי כתבות
- **Pollinations AI** (`text.pollinations.ai`) — מודל שפה לניתוח וסיכום (בחינם, ללא API key)

## 🚀 התקנה והרצה

### דרישות מוקדמות

- **Android Studio** — Hedgehog (2023.1.1) ומעלה
- **JDK 17**
- **Android SDK 34**
- **Min SDK:** 26 (Android 8.0)

### שלבי התקנה

1. **שכפל את הפרויקט:**
   ```bash
   git clone https://github.com/mastersauce/CrossNews.git
   cd CrossNews
   ```

2. **פתח ב-Android Studio:**
   - `File → Open` ובחר את תיקיית הפרויקט

3. **סנכרן את ה-Gradle:**
   - Android Studio יבצע `Sync` אוטומטית
   - אם לא — לחץ `Sync Now` בפס העליון

4. **הרץ על מכשיר/אמולטור:**
   - בחר מכשיר
   - לחץ על כפתור ה-Run (▶️)

### ללא API keys
האפליקציה לא דורשת מפתחות API — משתמשת בשירותים חינמיים וציבוריים.
